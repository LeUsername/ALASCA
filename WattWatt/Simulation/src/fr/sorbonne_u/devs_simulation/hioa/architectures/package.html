HIOA DEVS model descriptors for inclusion in architectures.

This package implements architectures of HIOA DEVS simulation models. In this library, architectures are descriptions of models that must be instantiated and connected in order to form a composed model ready to be run. The package fr.sorbonne_u.devs_simulation.architectures defines the overall architectures. This package defines the descriptions of atomic and coupled HIOA models that are included into overall architectures.
