Event scheduling view DEVS Simulation

The event scheduling world view is the standard simulation model used in stochastic discrete event simulations. See the following book for a very well-known reference.

A.M. Law, Simulation Modeling and Analysis, McGraw-Hill Int'l, 4th edition, 2007.

This view can be implemented within the DEVS protocol by merely adapting the implementation of its standard methods. The class AtomicES_Model gives such an implementation and its javadoc describes the choices made.

The implementation of event scheduling world view models in DEVS is presented in the following text available on the WWW:

  1. H. Vangheluwe, Discrete Event Modelling and Simulation, courseware, 2001.
